# bajty.pl

Prosta strona wizytówka projektu **bajty.pl**, oferującego gotowe komponenty integracyjne dla ekosystemu Java/Spring
Boot.

## O projekcie

**bajty.pl** to inicjatywa mająca na celu dostarczanie czystych, przetestowanych i gotowych do użycia komponentów
integracyjnych. Projekt narodził się z potrzeby uproszczenia uciążliwych integracji z publicznymi i przestarzałymi
usługami, które często napotykają programiści Java.

### Autor

**Mateusz Polak** – programista z ponad dziesięcioletnim doświadczeniem, pracujący wcześniej przy systemach rządowych,
medycznych oraz bankowych.

## Model działania

- **Open Source:** Podstawowe komponenty udostępniane za darmo na licencji MIT (np. TERYT Starter).
- **Wersje Premium:** Bardziej zaawansowane moduły planowane jako płatne rozwiązania wspierające rozwój projektu.

## Aktualne projekty

### TERYT Starter (W przygotowaniu)

Starter Spring Boot mapujący API SOAP (TerytWs1 Service) na czyste obiekty oparte o Java Records i enumy.
