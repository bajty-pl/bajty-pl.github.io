Nowoczesne i stabilne narzędzia integracyjne dla ekosystemu Java/Spring.

Eliminujemy ból integracji

Integracja aplikacji z polskimi systemami publicznymi (TERYT, REGON, KSeF) bywa drogą przez mękę. Naszym celem jest dostarczenie programistom czystych, nowoczesnych narzędzi, które ukrywają przestarzałe protokoły SOAP i WS-Security pod przyjaznym, wydajnym API.

teryt-spring-boot-starter

Open Source (MIT)

"Modern Spring Boot starter for seamless integration with the Polish national TERYT registry (GUS). Maps legacy SOAP/XML API into clean objects based on Java Records and enums."

Szybki start (Maven):

<dependency>
    <groupId>pl.bajty.teryt</groupId>
    <artifactId>teryt-spring-boot-starter</artifactId>
    <version>1.0.0</version>
</dependency>

Wersje Enterprise (Już wkrótce)

Dla projektów o znaczeniu krytycznym, E-commerce i systemów wymagających milisekundowych czasów odpowiedzi, przygotowujemy płatne moduły On-Premise dystrybuowane przez nasze prywatne repozytorium Nexus.

Local DB (Smart Synchro)

Automatyczne pobieranie paczek GUS (Spring Batch) i synchroniczny zapis do lokalnej bazy PostgreSQL/Oracle dla maksymalnej wydajności.

Autocomplete REST API

Gotowe punkty końcowe z obsługą wyszukiwania pełnotekstowego (Full-Text Search), literówek i formatowania pod frontend.
